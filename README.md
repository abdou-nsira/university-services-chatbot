# 🎓 AI University Chatbot

An AI-powered chatbot that answers student questions about university services using documents (PDFs).

## 🚀 Features
- Ask questions about exams, modules, deadlines
- Answers based only on uploaded documents
- Shows sources for transparency
- Supports PDF, TXT, MD files

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI API
- FAISS (vector search)
